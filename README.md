# e-plantShopping

This project is a shopping cart application for plants, built with React, Redux, and Vite.

## Overview

e-plantShopping provides a user-friendly interface for browsing and purchasing various plants. It demonstrates the use of modern web technologies to create a responsive and interactive shopping experience.

## Key Technologies

-   **React** for building the user interface.
-   **Redux** for state management.
-   **Vite** for fast development and building.
